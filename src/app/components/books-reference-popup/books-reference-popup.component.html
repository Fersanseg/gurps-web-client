Leyenda: Libros y códigos de referencia

En muchas partes de la página aparecen códigos que hacen referencia a alguna regla en concreto en un libro determinado, donde se puede consultar información adicional.

Por ejemplo: "L13" se referiría al libro "GURPS Lite", página 13. "B248" se refiere a "GURPS Basic Set", página 248.

  • B: GURPS Basic Set
  • L: GURPS Lite
