El carácter modular de las reglas de GURPS hace que sólo con el libro de reglas básico se pueda jugar una partida de cualquier género imaginable, desde fantasía hasta ciencia ficción, misterio, horror, etc. Eso significa que para jugar una partida en específico no sólo es recomendable, sino incluso obligatorio, ignorar parte de estas reglas básicas para evitar que el juego se vuelva demasiado pesado como para ser divertido.

En esta página se resumirán las principales reglas, junto con un código de referencia que indica la página del libro básico que explica dicha regla en mayor profundidad. Si no sabes cómo conseguir una copia del libro, dimelo y lo solucionaremos.
